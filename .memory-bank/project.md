# Project

Stake Planner

A professional bankroll planning application.

The purpose of this software is NOT to predict gambling outcomes.

The software ONLY calculates the optimal staking plan based on user-defined constraints.

This application should work for:

- Bingo18
- Keno
- Lottery
- Any fixed payout betting system

The application is purely a mathematical optimizer.

All calculations must be deterministic.

The application must never suggest that a strategy increases winning probability.

It only distributes bankroll.

Primary objective:

Find the minimum bankroll required while satisfying user constraints.
